## Adding a Project

To add a project to the README, append a new line under the `## Projects` section.

Each project entry must follow this format:

```
<emoji> [Project Name](https://github.com/dyc5828/<repo>) - One or two sentences describing what the project does.
```

Rules:
- One project per line
- Start with an emoji unique to that project (not reused by any other project in the list)
- Project name links to the GitHub repo
- Follow with a short description - one or two sentences max
- Keep descriptions focused on what the project does, not implementation details
